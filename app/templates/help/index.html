{# ── Overview ── #} {% call section('overview', 'Overview') %} {{ p('Mandate is a petition signature management system. It helps your team verify that petition signatures belong to registered voters in your target city, track progress toward your signature goal, and generate numbered petition books for collectors.') }} {{ h3('Roles') }}
Role Access
Enterer Home, Signature Entry, Collectors, Stats
Organizer Everything above, plus Users and Organizations
Admin Full access, including Settings, Voter Imports, and Print Books
{{ h3('Core Workflow') }}
  1. An admin uploads the voter file (registered voters in your jurisdiction).
  2. An admin generates numbered petition books and distributes them to collectors.
  3. Collectors gather signatures in the field.
  4. Enterers log in, start a session (select collector + book number), and enter each signature.
  5. For each signature, the enterer searches for the signer's address and records whether it matches a voter record.
  6. Stats pages show real-time progress toward the signature goal.
{% endcall %} {# ── Home & Sessions ── #} {% call section('home', 'Home & Sessions') %} {{ p('The Home page is your starting point for a data-entry shift. Before you can enter signatures, you must start a session that identifies which collector\'s book you are working on.') }} {{ h3('Starting a Session') }}
  1. On the Home page, select the collector whose book you are entering from the dropdown.
  2. Enter the book number (the serial number printed on the petition book).
  3. Click Start Session.
{{ tip('If the book number has already been used, the system will show you which collector it was previously assigned to. You can still start a session on it.') }} {{ h3('During a Session') }} {{ p('Once a session is active, your current collector and book number are shown at the top of the Home page. Navigate to Signature Entry to begin entering signatures.') }} {{ h3('Ending a Session') }} {{ p('Click End Session on the Home page when you have finished entering all signatures from a book. You can then start a new session for a different book.') }} {{ warn('Ending a session does not delete any entered signatures — it only clears the active book context so you can start a new one.') }} {% endcall %} {# ── Signature Entry ── #} {% call section('signatures', 'Signature Entry') %} {{ p('The Signature Entry page is where you verify each signature against the voter file. You must have an active session before you can enter signatures.') }} {{ h3('Searching for a Voter') }}
  1. Type the signer's street address into the search box (e.g., 123 Main St).
  2. Results appear automatically after three or more characters — no need to press Enter.
  3. Each result shows a registered voter's name, full address, and city.
{{ tip('Search by street number and street name only. You do not need to include apartment number, city, or zip code, though adding them can help narrow results.') }} {{ h3('Match Types') }}

After finding a result (or determining there is none), record one of three outcomes:

Person Match

The signer's name and address both match a voter record. This is a fully verified signature — it counts toward your goal.

Address Only

The address matches a voter record, but the name on the petition is different from the registered voter at that address. Record this when the address is valid but the name doesn't line up — it may still be reviewed later.

No Match

No voter is registered at the given address, or the address does not appear in the voter file at all. Record this so the signature is tracked even if it cannot be verified.

{{ p('After recording any match type, the form clears and is ready for the next signature on the same book.') }} {{ h3('Tips for Accurate Entry') }}
  • Use the address written on the petition — not what you think it should be.
  • If a search returns many results, add more of the address to narrow them down.
  • If someone lives in an apartment building, the street address alone may return the correct voter.
  • Abbreviations like "St", "Ave", "Blvd" generally work as well as full spellings.

Target City — the city whose registered voters count as "verified" signatures. Signatures from residents of this city are highlighted in stats and counted toward the goal. Select from the cities present in your voter file.
Signature Goal — the total number of verified signatures your campaign needs. Displayed as a progress bar on the Stats dashboard.
Allow PDF Deletion — when checked, admins can delete generated petition book PDFs from the Print Books page. Leave unchecked to prevent accidental or unauthorised removal of print records.
